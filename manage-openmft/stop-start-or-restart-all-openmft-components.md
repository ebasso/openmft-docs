# 4.2 Stop, Start or Restart all OpenMFT components

## 4.2.1 Change directory to base path

```text
cd $OPENMFT_BASE
```

## 4.2.2 Stop all OpenMFT components

```
om -stop all
```

## 4.2.3 Start all OpenMFT components

```text
om -start all
```

## 4.2.4 Restart all OpenMFT components

```text
om -restart all
```
