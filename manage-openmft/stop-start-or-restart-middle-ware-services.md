# 4.4 Stop, Start or Restart middle-ware services

## 4.4.1 Change directory to base path

```text
cd $OPENMFT_BASE
```

## 4.4.2 Stop middle-ware services

```
om -stop svc
```

## 4.4.3 Start middle-ware services

```text
om -start svc
```

## 4.4.4 Restart middle-ware services

```text
om -restart svc
```
